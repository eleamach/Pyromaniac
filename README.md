## Documentation format: 
All document name need to begin with `DOC-`, `SCHEMA-`, `MANUAL-`...
All document need to be placed in the **Doc** directory, and in the right sub-directory. 

## Git usage:
All your commits need to be compliant with [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/).
Use merge request, and it's not an advice, it's an order.
